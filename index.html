Earthquake Maps Animation Project by Amanda Hires


This software displays earthquakes from geodata on two maps:
- Google Terrain Map with animated markers and labels
- Google Standard Map with non-animated markers and labels

Click on the executable link for your platform to download and run the program. For system requirements and further instructions see the Readme File.
